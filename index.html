Basic Table Structure

Defining a Table

<table></table>

To get started, we have to define our table in HTML. We do this by using the <table></table> tags. All other information relating to this table must then be placed between these tags.

<thead></thead>

To make our code more readable, we use the semantic tags <thead>, <tbody> and <tfoot> to split out the different sections of our table. The first of these is the table header, <thead>, which sections the first row(s) of our table which will contain the headings for each column.

Within the <thead> tags we will then define the row(s) and the cells for that row:

We do this using the table row, <tr>, and table header, <th>, tags.

Heading 1 Heading 2 Heading 3

This is a good start! It doesn't really look like a table yet, so let's look at adding some more rows!

<tbody></tbody>

Now that we have set up the headers for our columns, we need to add cells below the headings to hold the relevant data.

We do this by creating a body, <tbody>, section and defining more rows within the body of the table using the <tr> tags.

The thing we do slightly differently this time, however, is to define each cell using the table data, <td>, tags. These tags tell the brower that the cells are holding data rather than headings (which would be defined using the previous <th> tag).

Now this looks more like a table! Let's add the final section!

<tfoot></tfoot>

The final section of a table in HTML is called the table footer and is enclosed with the <tfoot> tags.

Many tables utilise the final row(s) to sum data in each column or provide quick summaries for each column.

Just like in the previous section of the table, we use the <tr> and <td> tags to define each row and cell.

Summary

Let's summarize what we've learnt:

Element HTML Tag Description
Table <table> Defines the table. All other code for the table is enclosed in these tags
Table Row <tr> Defines a row within the table
Table Heading <th> Defines each cell within the <thead> section
Table Data <td> Defines each cell within the rest of the table
Table Header <thead> Semantic tag to show the headings section
Table Body <tbody> Semantic tag to show the body of the table
Table Footer <tfoot> Semantic tag to show to final row(s) of the table
Column Span <td colspan=""> Makes a cell span the specified number of columns
Row Span <td rowspan=""> Makes a cell span the specified number of rows
